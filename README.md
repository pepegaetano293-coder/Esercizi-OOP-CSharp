# Percorso di Apprendimento OOP in C#
Questa reposity contiene una serie di esercizi pratici focalizzati sui pilastri della Programmazione Orientata agli Oggetti (OOP) in C#. Il percorso parte dalle basi dei costruttori fino alla gestione di interazioni complesse tra oggetti.

# Concetti Appresi
**Costruttori e Chaining:** Utilizzo di ": this()" per evitare la duplicazione del codice e delegare l'inizializzazione tra costruttori.
**Incapsulamento:** Protezione dei dati tramite campi privati (_saldo) e proprietà pubbliche con logica di controllo nei "get" e "set".
**Composizione ("Has-a"):** Gestione di liste di oggetti all'interno di altre classi (es. una "Playlist" che contiene una "List<Canzone>").
**Ereditarietà ("Is-a"):** Creazione di gerarichie tra classi Madre e classi Figlie (es. Manager: Dipendente)
**Metodi di Istanza:** Creazione di comportamenti specifici per gli oggetti (es. ApplicoSconto, PrestitoLibro).
**Comunicazione tra Oggetti:** Come un oggetto può influenzare lo stato di un altro (es. il sistema di combattimento tra Personaggi).

# Tecnologie usate
**Linguaggio:** C#.
**Ambiente di Sviluppo Integrato:** Visual Studio.
